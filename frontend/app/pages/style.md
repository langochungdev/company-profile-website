# Style Guide - Company Profile Website

## Màu sắc chủ đạo

-   **Primary (Đỏ):** #DC2626 - Màu logo công ty
-   **Secondary (Đen):** #111111 - Tạo sự chuyên nghiệp, mạnh mẽ
-   **Light Mode:** Nền trắng #FFFFFF, chữ đen/xám đậm

## Phong cách thiết kế

-   **Thiên hướng:** Xây dựng, kỹ thuật, công nghệ
-   **Ngành nghề:** Lắp đặt camera an ninh
-   **Tone:** Chuyên nghiệp, đáng tin cậy, hiện đại

## Bảng màu chi tiết

| Tên             | Mã màu  | Mục đích                      |
| --------------- | ------- | ----------------------------- |
| primary         | #DC2626 | Button, accent, logo          |
| primary-dark    | #B91C1C | Hover states                  |
| primary-light   | #EF4444 | Highlight                     |
| secondary       | #111111 | Text heading, background dark |
| secondary-light | #1F1F1F | Hover dark                    |
| accent          | #F87171 | Decoration                    |

## Typography

-   **Font chính:** Inter (sans-serif)
-   **Font display:** Playfair Display (serif)

## UI Components

-   Border radius: rounded-lg (8px)
-   Shadow: shadow-lg cho cards
-   Transition: 300ms cho hover effects
